2. Welche Daten verarbeitet der Bot?

Der Bot verarbeitet nur die Daten, die technisch notwendig sind, um seine Funktionen auszuführen. Dazu können gehören:

2.1 Automatisch verarbeitete Daten

• Discord‑User‑IDs
• Discord‑Server‑IDs
• Kanal‑IDs
• Nachrichteninhalte (nur, wenn der Bot darauf reagieren muss)
• Rolleninformationen
• Interaktionsdaten (Slash‑Commands, Buttons, Reaktionen)


2.2 Gespeicherte Daten (falls zutreffend)

Der Bot speichert nur Daten, die für bestimmte Funktionen notwendig sind, z. B.:

• Einstellungen des Servers (Prefix, Moderationsregeln, Rollen)
• User‑Einstellungen (z. B. Levelsystem, XP, Profile)
• Logs (nur wenn aktiviert)


Es werden keine Daten dauerhaft gespeichert, die nicht für die Funktion des Bots notwendig sind.

---

3. Zweck der Datenverarbeitung

Der Bot verarbeitet Daten ausschließlich zu folgenden Zwecken:

• Ausführen von Befehlen
• Moderation (z. B. Auto‑Mod, Logging)
• Server‑Management
• Unterhaltung (z. B. Musik, Games, Levelsystem)
• Verbesserung der Bot‑Funktionen


Es findet keine Weitergabe der Daten an Dritte statt.

---

4. Rechtsgrundlage

Die Verarbeitung erfolgt gemäß:

• Art. 6 Abs. 1 lit. b DSGVO – Vertragserfüllung (Bereitstellung der Bot‑Funktionen)
• Art. 6 Abs. 1 lit. f DSGVO – berechtigtes Interesse (Sicherheit, Stabilität, Missbrauchsschutz)


---

5. Speicherdauer

Der Bot speichert Daten nur so lange, wie sie für die jeweilige Funktion notwendig sind.

Beispiele:

• Temporäre Daten → sofort gelöscht
• Server‑Einstellungen → bis der Server sie löscht
• User‑Daten → bis der User sie löscht oder den Server verlässt


---

6. Weitergabe von Daten

Es werden keine Daten an Dritte weitergegeben.
Es findet kein Tracking, kein Profiling und keine Werbung statt.

---

7. Rechte der Nutzer

Nutzerinnen und Nutzer haben folgende Rechte:

• Auskunft über gespeicherte Daten
• Löschung ihrer Daten
• Berichtigung falscher Daten
• Einschränkung der Verarbeitung
• Widerspruch gegen die Verarbeitung


Anfragen können über den Kontakt oben gestellt werden.
